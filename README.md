A classic Snake Game built using **Python** and the **Turtle graphics module**.  
The game includes food generation, score tracking, snake growth, and collision detection.

- Control the snake using arrow keys
- Random food appears on the screen
- Snake grows after eating food
- Score increases when food is eaten
- Game over when snake hits the wall
- Game over when snake collides with itself
- save the highest score.
  

**Concepts Used:**

- Python programming
- Object-Oriented Programming (OOP)
- Turtle graphics
- Keyboard event handling
- Game loop and collision detection
- Store the data after run the code
